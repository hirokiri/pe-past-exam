# 技術士過去問題模範解答解説集　要件定義

## システムの概要

このシステムは[技術士試験公式WEBサイト](https://www.engineer.or.jp/sub02/)に公開されている技術士試験の過去問題について、その問題と模範解答及び解説を網羅したWebアプリケーションである。

Webアプリは 検索・絞り込み UI（部門・年度・キーワード検索）を可能とする。

また作成したコンテンツは下記のフォーマットで出力できるものとする。

- html
- pdf
- epub

出力単位は、部門ごととする

過去問題の範囲は、下記のとおりとする。

- 技術士第一次試験（基礎科目・専門科目）
  - [公式ページ](https://www.engineer.or.jp/c_categories/index02021.html)に掲載の全範囲（2026/07/06 現在）
  - 科目:
    - 基礎科目
    - 適性科目
    - 専門科目（01機械～20原子力・放射線部門）
  - 年度： 平成23年度～令和7年度
- 技術士第二次試験
  - [公式ページ](https://www.engineer.or.jp/c_categories/index02022.html)に掲載の全範囲（2026/07/06 現在）
  - 全部門（01機械部門　～　21総合技術監理部門）
    - 必須科目
    - 選択科目
    - (現在実施していない科目は省略)

まず当初の実装は、第二次試験 > 01 機械部門 > 必須科目, 選択科目: 0103 機構ダイナミクス・制御 の令和元年～令和7年度までとする。

### 記載の順序

1つの問題ごとに

- 問題　→　模範解答　→　解説

のように直ちに正答内容を確認できる構成とする。

## 模範解答とその解説

- できるだけ最新の情報を信頼できるWebページや論文等の文献から調査すること。
- 簡潔かわかりやすく記述すること
- 模範解答・解説を生成パイプラインを作って段階的に拡充する
- まず「特定の部門・年度でMVPを作り、横展開する」フェーズ分けを認める
- 解説の品質基準
  - 概ね1ページに収まる分量とする。
  - 出典の引用を明示すること。
  - 複数の文献にあたり、セルフレビューを行うこと

## 問題データの取得・構造化方法

  公式サイトの過去問は PDF（図・数式・表を含む）である。

- 図表や数式についてはMermaidやLatexで作成できる場合は、作成し困難な描画や写真の場合はPDFから切りだすこと
- 二次試験の「模範解答」は記述式論文（指定されている文字数 600〜1800字など）を意味する
- コンテンツMarkdownファイルは直接リポジトリ内に保存するものとする。
