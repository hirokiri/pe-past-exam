# 技術士過去問題模範解答解説集　要件定義

## システムの概要
このシステムは[技術士試験公式WEBサイト](https://www.engineer.or.jp/sub02/)に公開されている技術士試験の過去問題について、その問題と模範解答及び解説を網羅したWebアプリケーションである。

また作成したコンテンツは下記のフォーマットで出力できるものとする。

 - html
 - pdf
 - epub

過去問題の範囲は、下記のとおりとする。
- 技術士第一次試験（基礎科目・専門科目）
  - [公式ページ](https://www.engineer.or.jp/c_categories/index02021.html)に掲載の全範囲（2026/07/06 現在）
  - 科目: 
    - 基礎科目
    - 適正科目
    - 専門科目（01機械～20原子力・放射線部門）
  - 年度： 平成23年度～令和7年度
- 技術士第二次試験
  - [公式ページ](https://www.engineer.or.jp/c_categories/index02022.html)に掲載の全範囲（2026/07/06 現在）
  - 全部門（01機械部門　～　21総合技術監理部門） 
    - 必須科目
    - 選択科目
    - (現在実施ていない科目は省略)

### 記載の順序
1つの問題ごとに

- 問題　→　模範解答　→　解説

のように直ちに正答内容を確認できる構成とする。

### 模範解答とその解説
模範解答とその解説についてはできるだけ最新の情報を信頼できるWebページや論文等の文献から調査し、簡潔かわかりやすく記述すること
